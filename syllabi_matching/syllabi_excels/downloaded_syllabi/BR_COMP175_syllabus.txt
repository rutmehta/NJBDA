DETAILED COURSE DESCRIPTION

This course teaches the student the fundamental concepts needed to design 
and develop a game. The focus is on the development process and the 
documentation required to successfully implement a game, including the 
creation of a game treatment and game spec. Rules, play mechanics, balance, 
storytelling, interface design, and team management, and differentiation 
between genres are also covered.

SCHEDULE OF TOPICS TO BE COVERED

UNIT 1: Introduction to Game Design



The student will be able to:

Define game design

Identify design documents

Understand the skills required of a game designer

Develop an idea for a game

Understand elements which influence game design



Assignments:

Read Chapters 1 and 2 in text.

Assignment 1: Complete game concept worksheet on page 53.  Submit an idea 
for an original game. 



The student will be able to:

Create a world for a game

Define a games concepts of time, space, emotions, and logic.

Differentiate between realism and abstraction

Develop a story

Create a narrative

Assignments:

Read Chapters 3, 4 and Appendix A in text.

Assignment 2: Complete worksheet on page 118.

Project 1: Create a High-Concept Document for a game.
